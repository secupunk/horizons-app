# Horizons App

Application web créée avec React + Vite.

## Déploiement sur Vercel

Ce projet est configuré pour être déployé sur Vercel.

### Variables d'environnement requises

Configurez ces variables dans Vercel :
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_PUBLISHABLE_KEY`

## Développement local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
